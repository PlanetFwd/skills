# SKILLS

This repository is a collection of SKILLS.md that we have collected at Planet
FWD to support our team and customers. Please feel free to use, improve, and
contribute to better SKILLS.md

## Usage:

1. Clone this repository

2. Run `./compile-skills.sh`

3. Follow instructions from your favorite LLM to install skills.
